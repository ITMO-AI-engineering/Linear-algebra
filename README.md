## Лабораторная работа: Метод главных компонент (PCA)

### Описание
Реализация алгоритма PCA (Principal Component Analysis) с нуля на Python. Проект включает:
- Теоретическое обоснование метода
- Реализацию ключевых этапов PCA
- Визуализацию результатов

### Основные компоненты
1. **Теоретическая часть**:
   - Математическое обоснование PCA
   - Доказательство оптимальности главных компонент

2. **Практическая реализация**:
   - Класс `SparseMatrixCRS` для работы с разреженными матрицами
   - Метод Гаусса для решения СЛАУ
   - Функции для:
     - Центрирования данных
     - Вычисления матрицы ковариаций
     - Нахождения собственных значений и векторов
     - Проекции данных на главные компоненты

3. **Визуализация**:
   - Сравнение исходных данных и их проекции
   - Анализ объясненной дисперсии
